# Get Quality Metrics

Use `mcp__curation_server__get_quality_metrics` to view KB health.

## Metrics Returned

- Total solutions count
- Solutions by status (PROPOSED, VERIFIED, PROVEN, etc.)
- Average success rate
- Solutions needing review
- Top agents by contribution
- Recent activity

## User Input

```text
$ARGUMENTS
```

Call `mcp__curation_server__get_quality_metrics` and display results.
