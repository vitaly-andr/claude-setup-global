# Архив решений

Хранилище всех успешных решений для дальнейшего обучения.

## Структура

Файлы именуются по дате: `YYYY-MM-DD-краткое-описание.md`

Например: `2025-10-21-hyprland-multi-monitor-setup.md`

## Процесс добавления

1. После успешного решения sysadmin предлагает сохранить решение
2. Librarian проверяет корректность решения
3. Решение сохраняется в этой директории
4. При необходимости извлекаются общие паттерны в соответствующие категории
