# Claude Code Knowledge Base

Централизованное хранилище проверенных решений и команд для агентов Claude Code.

## Структура

```
knowledge/
├── README.md              # Этот файл
├── METHODOLOGY.md         # Методика пополнения и использования
└── sysadmin/             # База знаний для sysadmin агента
    ├── arch-linux/       # Специфичные команды и решения Arch Linux
    ├── hyprland/         # Конфигурации и решения Hyprland
    ├── waybar/           # Конфигурации и решения Waybar
    ├── common-issues/    # Часто встречающиеся проблемы и решения
    └── solutions/        # Архив успешных решений
```

## Принципы

1. **Проверенные решения**: Только успешно протестированные команды и конфигурации
2. **Актуальность**: Указываем дату и версию ПО
3. **Структурированность**: Единый формат для всех записей
4. **Постепенное пополнение**: Добавляем после каждого успешного решения сложной задачи

## Формат записей

См. METHODOLOGY.md для детального описания процесса.
